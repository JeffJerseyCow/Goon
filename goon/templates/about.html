About Goon

Goon is a Keyword association engine that utilises the power of your web broswer to quickly navigate around localised networks. Simply create a new search engine entry using the following settings:

Search Engine
Goon
Keyword
go
Goon String
http://goon/query?kw=%s

Then type go keyword into your search bar. If you'd like to create a new link association visit the homepage and enter your desired Keyword/URL combination.
